INCAR
=====
INCAR各个选项的汇总，可以作为编写INACR文件的参考。

A generic INCAR file for the density functional theory package VASP. 

Requirements
------------
A VASP license, and the other required input files (POSCAR; KPOINTS and POTCAR)
